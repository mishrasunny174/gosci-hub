# GOSCI-HUB
A simple cli application to download pdf of research papers from sci-hub written in go

## Installation

```
go instal github.com/mishrasunny174/gosci-hub
```

## Usage

```
âžœ gosci-hub -h
Usage of gosci-hub:
  -outfile string
    	Output File (default "output.pdf")
  -url string
    	Link of article you want to download
```
